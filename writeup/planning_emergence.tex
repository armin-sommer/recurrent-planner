\large Binding, transition-respecting edges, and amortized policy evaluation on Sokoban}
\author{Recurrent-Planner project}
\date{\vspace{-0.6em}\today}

\begin{abstract}
\noindent Model-free RL on a state-indexed relational recurrent core induces \emph{planning}, and the
cleanest description is a \textbf{graph}: the recurrent attention turns each board into a weighted directed
graph $G{=}(V,A)$ over its squares, and the ``thinking'' loop plans by \textbf{propagating value along $G$}.
With \emph{no} locality mask we show a three-step chain: (1)~the graph's \textbf{nodes bind} to environment
states; (2)~its \textbf{edges follow} the board's one-step move graph $\NN$ (each node links to its walkable
neighbours, edge weight decaying with graph distance, ${\sim}0$ across walls); and (3)~the loop runs
\textbf{amortized policy evaluation} on $G$ --- iterating the graph operator to spread value from the goal
node, $c{\leftarrow}\gamma_{\mathrm{eff}}Ac{+}r$ (no $\max$: a convex average over each node's edges; the
improvement is a one-step lookahead at the head). All shown causally on held-out levels of a 300M-step
Sokoban model.
\end{abstract}

\vspace{-0.6em}
\section*{Setup --- the core as a graph}
\vspace{-2pt}
A state-indexed attention-LSTM turns each board into a weighted directed graph $G{=}(V,A)$. The
\textbf{nodes} $V$ are the $S{=}H{\times}W{=}100$ board squares, each carrying a $C{=}32$-dim latent $h(s)$.
Per ``thinking'' tick the attention assigns an \textbf{edge weight} $A_{ss'}$ --- how much node $s$
aggregates from node $s'$ --- computed from the nodes' contents by fixed weights and row-normalised
($\textstyle\sum_{s'}A_{ss'}{=}1$; dense, no locality mask; 4 heads; sparse 1.5-entmax). A tick aggregates
neighbour latents along these edges and applies an LSTM gate; the loop runs up to $K{=}6$ ticks ($D{=}3$
stacked cells). Because $A$ is computed from content, it is \textbf{re-derived per board} --- one fixed rule,
a board-specific graph. We write $\NN$ for the board's true one-step move graph. Trained 300M IMPALA steps.

\section{Binding: the nodes are the states}
\vspace{-2pt}
Each node encodes the environment state at its square: a linear probe recovers the tile type from its latent
$h(s)$ well above chance for every object class, including the rare ones (box/target/agent are $1$--$4\%$ of
squares), and this is flat across ticks --- read from the input embedding, not built by the loop. The graph's
node features \emph{are} the states.

\section{The learned graph: edges follow $\NN$}
\vspace{-2pt}
Do the learned \textbf{edges} trace the board's move graph $\NN$? Yes. Each node's edge weight concentrates
on its (wall-masked) walkable neighbours at $7$--$13\times$ chance, and edge weight decays geometrically with
the \emph{graph} distance between nodes (Fig.~\ref{fig:kernel}: $\rho{\approx}0.66$, reach $\sim$3 hops,
${\sim}0$ weight to graph-unreachable nodes, plus a small $+0.07$ edge to the goal node). The structure is
\emph{causal} and graph- (not pixel-) defined: at matched pixel distance, a node separated from the agent by
a wall (graph-far) influences it only $0.38\times$ as much as a graph-near one (through-walls test; partial
corr.\ of influence with graph-distance, controlling Euclidean, $=-0.32$). Since the edges are re-derived
from content, each board gets its \emph{own} graph tracing \emph{its} walls --- the same fixed rule, a
board-specific adjacency. What \emph{flows} on this graph is \S3.

\section{Planning: value propagation on the graph}
\vspace{-2pt}
Planning is a \emph{dynamics on $G$}: a value sits on each node and is propagated along the edges.
Linearising one weight-tied tick gives it in closed form,
\vspace{-2pt}
\[
c \;\leftarrow\; \gamma_{\mathrm{eff}}\,A\,c + r
\quad\Longrightarrow\quad
c^\star=(I-\gamma_{\mathrm{eff}}A)^{-1}r=\sum_{i\ge0}(\gamma_{\mathrm{eff}}A)^{i}\,r,
\]
which is value iteration / \textbf{policy evaluation on the graph}: $A$ is the graph operator
(${\approx}\gamma P^{\pi}$), $\sigma(g_f)$ the discount, and $r$ the source injected at the goal node. The
aggregation $z=(A v)W_{\mathrm{out}}=\mathbb{E}_{s'\sim A}[\cdot]$ is a \emph{convex average} over a node's
edges --- no $\max$ --- so the loop \emph{evaluates}; the $\max$ (improvement) is a single
$\arg\max_a[r_a+\gamma V(s'_a)]$ at the actor head.

The propagation is \emph{multi-step and deepens with thinking}: a perturbation $d$ edges from the agent
reaches its node only after several ticks, building from $\approx0$ (Fig.~\ref{fig:reach}); one tick spans
$\sim$3 edges, yet nodes $7$--$9$ away still arrive via intermediate relays --- reach \emph{compounds} along $G$. The converged value is a recursive Bellman fixed point (optimality
residual$/\mathrm{std}(V)$ low and flat over greedy successors $0$--$3$), and the policy increasingly matches
one-step value-lookahead as it thinks ($0.34{\to}0.41$, chance $0.25$).

\paragraph{The value tracks edits to the source and the graph, and re-plans.} Causal interventions confirm a
genuine policy-evaluation field (Fig.~\ref{fig:abl}). Moving the goal (the \emph{source} node) shifts the
value in the resolvent-predicted direction ($\mathrm{corr}(\Delta V,-\Delta\text{dist}){>}0$, a box move
more). Adding a wall \emph{edits the graph}: a wall that
lengthens the agent's path lowers the value ($-1.3\sigma$ vs $-0.35\sigma$ for an off-path wall), the
effect \emph{builds over ticks}, reaches the agent's \emph{own} node ($2.2\times$ off-path) and re-plans its
move ($3\times$ a cosmetic wall). The value re-forms by propagating the consequence as \emph{content} along
$G$, not by re-routing edges. Reading the model's \emph{own}
head per tick confirms the loop drives the \emph{decision}: the action \emph{changes} on $35\%$ of boards and
grows more goalward as value converges ($0.52\!\to\!0.60$, chance $0.25$), the gain largest where the goal is
farthest (Fig.~\ref{fig:decide}, $+0.15$ at distance ${\ge}13$) --- the converging value re-ranks the head's logits.

\vspace{-4pt}
\section*{Conclusion}
\vspace{-2pt}
Planning emerges from RL as a \emph{dynamics on a learned graph}: attention binds \textbf{nodes} to states
(\S1), shapes \textbf{edges} along the move graph $\NN$ (\S2), and the loop \textbf{propagates value from the
goal node} --- policy evaluation with a one-step lookahead at the head (\S3). Not value iteration (no $\max$),
a successor representation (source in the iterate), or tree search (only the immediate action decodes).

\section{Architecture and training}
\vspace{-2pt}
\textbf{Core (graph form).} A state-indexed attention-LSTM. The nodes are the $S{=}H{\times}W{=}100$ board
squares; the per-tick attention $A$ (below) is the learned weighted adjacency over them. The recurrent carry
is, per stacked cell $d\in\{0,1,2\}$ ($D{=}3$), a cell memory $c_d$ and hidden $h_d$, each in
$\mathbb{R}^{S\times C}$ ($C{=}32$); $x\in\mathbb{R}^{S\times C}$ is a convolutional board embedding, fixed
across ticks. Each tick applies, for $d=0,1,2$ bottom-to-top (with $\mathrm{prev}$ the hidden of the cell
below, or the top hidden from the previous tick for $d{=}0$), the weight-tied update
\vspace{-3pt}
\begin{align*}
u &= \mathrm{RMSNorm}\!\big(h_d + [x;\mathrm{prev}]W_{\mathrm{in}}^d + b^d\big)\odot s^d,\\
q,k,v &= u W_q^d,\; u W_k^d,\; u W_v^d \quad(\text{4 heads},\ d_h{=}8),\\
A &= \mathrm{entmax}_{1.5}\!\big(qk^{\top}/\sqrt{d_h} + \mathrm{RelBias}^d\big),\qquad z=(A\,v)\,W_{\mathrm{out}}^d,\\
[g_i,g_j,g_f,g_o] &= [x;\mathrm{prev};z]\,W_g^d + b_g^d,\\
c_d &\leftarrow c_d\,\sigma(g_f) + \tanh(g_i)\,\sigma(g_j),\qquad h_d \leftarrow \tanh(c_d)\odot\tanh(g_o).
\end{align*}
\vspace{-6pt}

$A_{ss'}$ are the edge weights (row-stochastic; dense, no locality mask; sparse via 1.5-entmax); the
aggregation $z=(A v)W_{\mathrm{out}}$ is a convex average over a node's edges (no $\max$). Up to $K{=}6$
ticks. The per-board readout pools $m=\mathrm{ReLU}\big((h_2+x)W_{\mathrm{dense}}+b\big)$ over the nodes into
actor logits $mW_a+b_a$ and a critic value $V=(mW_c+b_c)\,s_{\mathrm{head}}$.

\textbf{Training.} IMPALA on Boxoban unfiltered-train, $300$M steps, $\gamma{=}0.97$, variable thinking
depth $d\sim\mathcal{U}\{1..6\}$ per rollout (the learner replays the same depth). Online train solve
$\approx83\%$, \texttt{valid\_medium} $\approx31\%$. All probes use the held-out \texttt{valid\_medium}
split at the $300$M checkpoint.

\textbf{Faithful recompute.} The model's own forward hits a tracer error on the offset-tied
\texttt{rel\_bias} gather under \texttt{nn.scan}; every probe instead recomputes the $D{=}3$ entmax stack in
plain JAX from the loaded parameters (\texttt{recompute\_d3}), validated to reproduce the model's final
hidden state to $1.8\times10^{-7}$.

\section{Experiments}
\vspace{-2pt}
Scripts are under \texttt{results/}; all on held-out \texttt{valid\_medium} at the $300$M checkpoint.

\textbf{Step 1 --- binding (the nodes are the states).}
\begin{itemize}[leftmargin=1.4em,itemsep=1pt,topsep=2pt]
\item \textbf{Binding probe} (\texttt{interp\_planning\_d3.py}): linear probe $h(s)\!\to\!$ tile type,
per-object balanced accuracy (chance $0.5$) --- wall/box/target/agent $=0.77/0.76/0.70/0.75$, flat over ticks.
\end{itemize}

\textbf{Step 2 --- the learned graph (edges follow $\NN$).}
\begin{itemize}[leftmargin=1.4em,itemsep=1pt,topsep=2pt]
\item \textbf{Recovery of $\NN$} (\texttt{interp\_planning\_d3.py}): edge weight on the wall-masked one-step
move-neighbours vs uniform chance --- $7$--$13\times$; entmax support broad ($\sim$44--88 keys per node).
\item \textbf{Through-walls} (\texttt{interp\_wall\_d3.py}): flip a floor node to a wall at matched pixel
distance; agent-latent influence for graph-far (blocked) vs graph-near (open) nodes --- blocked
$=0.38\times$ open, partial corr$(\text{infl},\text{graph}\!\mid\!\text{Euclid})=-0.32$.
\item \textbf{Learned kernel} (\texttt{interp\_e4\_d3.py}): edge weight vs graph distance --- geometric decay
$\rho{\approx}0.66$ (reach $\sim$3 hops), $34\%$ of weight beyond one hop, $0.00$ to graph-unreachable nodes,
$+0.07$ directly to the goal node.
\end{itemize}

\textbf{Step 3 --- value propagation on the graph.}
\begin{itemize}[leftmargin=1.4em,itemsep=1pt,topsep=2pt]
\item \textbf{No-max / static operator} (config + \texttt{interp\_e1\_d3.py}): trained config is
\texttt{readout="softmax"} (convex average), \texttt{maxplus} off; the edges $A$ are stationary and do not
sharpen (top-1 mass ${\approx}0.16$, ${\sim}85$ keys, $\cos(A_t,A_{t-1})\!\to\!0.995$) $\Rightarrow$ evaluation, not iteration.
\item \textbf{Reach compounds} (\texttt{interp\_e5\_d3.py}): perturb a node $d$ edges away; agent-influence
onset per tick --- far nodes ($d{=}7$--$9$) arrive only after ${\sim}3$--$4$ ticks (from ${\approx}0$); horizon grows.
\item \textbf{Bellman self-consistency} (\texttt{interp\_bellman\_d3.py}): optimality residual
$|V-\max_a(r{+}\gamma V(s'))|/\mathrm{std}(V)$ at agent and greedy successors $0$--$3$ $=[0.19,0.25,0.25,0.22]$
(low, flat) $\Rightarrow$ recursive fixed point.
\item \textbf{Lookahead-consistency} (\texttt{interp\_lookahead\_d3.py}): policy agreement with
$\arg\max_a[r_a{+}\gamma V(s'_a)]$ over its own value $=0.41$ (chance $0.25$), rising $0.34\!\to\!0.41$ with depth.
\item \textbf{Reward-relabel} (\texttt{interp\_e2\_d3.py}): move the goal/source (keep edges) vs a matched
wall-flip; far-node value shift $0.35$ vs $0.22$ (ratio $1.55$) $\Rightarrow$ source in the iterate
(policy evaluation, not a successor representation).
\item \textbf{Far-side interventions} (\texttt{interp\_e6b\_d3.py}): relocate goal/box far ---
$\mathrm{corr}(\Delta V,-\Delta\text{dist})=+0.18/{+}0.16$; box impact $1.5\sigma>$ goal $1.0\sigma$.
\item \textbf{Edge adaptation} (\texttt{interp\_e7\_d3.py}, \texttt{interp\_e7b\_d3.py}): the edges $A$ are
near-invariant to a single injected edit (row-change near/far ratio $1.1\times$) $\Rightarrow$ no re-wiring;
and \textbf{node content} (\texttt{interp\_e8\_d3.py}) value-projection norm wall/floor $=0.94$ (flip ratio
$1.06$) $\Rightarrow$ wall nodes are not ``dead''.
\item \textbf{New physics, reaches the agent + re-plans} (\texttt{interp\_e9\_d3.py},
\texttt{interp\_e10\_d3.py}): a path-blocking wall lowers the value ($94\%$, $-1.3\sigma$) vs an off-path wall
($-0.35\sigma$), the effect builds over ticks, shifts the agent's \emph{own} node latent $2.2\times$ off-path,
and changes its greedy action $3\times$ more often $\Rightarrow$ the value adopts the new graph and re-plans.
\item \textbf{Decision over ticks} (\texttt{interp\_e13\_d3.py}): reading the model's own head per tick (512
boards), the action changes on $35\%$ of boards and grows more goalward ($0.52\!\to\!0.60$, chance $0.25$;
per agent$\to$goal distance band $+0.10/{+}0.07/{+}0.08/{+}0.15$, largest on the farthest) as value contracts
($|\Delta V|\,0.72\!\to\!0.23$) and the decision margin sharpens ($1.6\!\to\!3.3$).
\end{itemize}

\noindent Together: the loop propagates a goal-sourced value field along a fixed, transition-respecting graph
(policy evaluation); the value tracks edits to the source (goal/box) and the graph (walls), reaches the agent
node, and re-plans; and the decision grows more goalward as that value converges --- propagating value
\emph{content}, not re-wiring edges.
